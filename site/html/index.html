Restbed

Developing RESTful Web services that seamlessly support exposing your data in a variety of representation media types and abstract away the low-level details of the consumer-service communication is not an easy task without a good framework. In order to simplify development of RESTful Web services in C++, a standard and portable Restful API has been designed.

Features

  • HTTP Method, Path, and Content-Type request filteration
  • Path parameters
  • C++11 Support
  • Linux, BSD, Mac OSX, and Windows platform support
  • Two way SSL certificate authentication
  • Designed with inheritance and customisation in mind
  • Logging
  • Thread Safe
  • Logging
