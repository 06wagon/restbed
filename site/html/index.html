Developing RESTful Web services that seamlessly support exposing your data in a variety of representations while abstracting away the low-level details of the HTTP protocol is not an easy task without a good framework. Restbed is an attempt to simplify the development of such systems within the C++ development arena. It offers a clean, portable interface to get RESTful Web Services up and running within a matter of minutes.

Features

  • HTTP Method filteration
  • Path Parameters
  • Header request fliteration
  • Latest C++11 Feature Support
  • Linux, BSD, Mac OSX, and Windows platform support
  • Two way SSL certificate authentication
  • Designed with developer customisation in mind
  • Built in logging functionality
  • Thread Safe
